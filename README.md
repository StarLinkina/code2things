# code2things-zhou-tao

## 项目简介

这是一个静态网页项目（示例/练习/作品集），用于记录课堂学习与个人前端练习。仓库包含若干示例页面、样式文件、图片和视频资源，便于学习和展示小型前端成果。

## 目录结构（主要）

- `html/`：站点页面入口和示例页面（含 `index.html`）。
- `css/`：样式表（包含 Bootstrap 样式文件）。
- `js/`：脚本文件（包含 Bootstrap 脚本）。
- `images/`：图片资源。
- `project_html/`：项目示例页面（如 `Project1.html`, `Project2.html`）。
- `weekly_html/`：按周整理的练习或课堂笔记页面。
- `videos/`：课程或演示视频资源。

## 本地运行

此项目为纯静态文件，可直接在浏览器中打开 `html/index.html`。若需要通过本地 HTTP 服务运行，可使用 Python 或其他轻量级静态服务器：

```
python -m http.server 8000
# 在浏览器打开 http://localhost:8000/html/index.html
```
