黑客松项目日志

项目简介

本项目设计了一款基于 ESP32-C3 微控制器的可穿戴久坐提醒设备。 通过 MPU-6050 六轴惯性传感器实时检测用户身体运动状态， 搭配 SSD1306 OLED 显示屏直观显示当前持续久坐时间， 并在超过预设时长后给出视觉提示。 设备采用纯 ASCII 字符显示，无额外 LED 模块， 结构简洁、功耗低，适合嵌入式穿戴场景。

功能介绍

功能 介绍
运动状态检测 利用 MPU-6050 三轴加速度计采集加速度数据；
计算连续采样间的加速度矢量变化幅度；
当变化幅度超过可配置阈值时判定为“起身活动”，自动将久坐计时清零。
久坐计时与显示 当加速度连续稳定若干次后，判定用户“已坐定”，开始累加计时；
SSD1306 OLED 实时显示“已坐时间”（分:秒格式），最大显示 99 分 59 秒；
分级状态提示
  • 正常：久坐时间 < 警告阈值 → 显示 Status: OK
  • 警告：久坐时间 ≥ 警告阈值 → 显示 Status: Time 2 move
  • 严重：久坐时间 ≥ 严重阈值 → 显示 Status: TOO LONG! >_<
可调节参数 代码顶部集中定义所有可调参数，用户可方便修改

设计运动检测算法

设定 200ms 采样间隔，读取加速度 (ax, ay, az)；

与上一次采样值求差，计算三维向量模长；

若模长 > ACCEL_THRESHOLD（默认 3.0 m/s²），认为用户处于运动状态；

引入连续稳定计数，避免短暂抖动导致误判。

总结

本项目成功实现了一款体积小巧、反应灵敏的久坐提醒穿戴设备。 通过 ESP32-C3 + MPU-6050 + SSD1306 的组合， 利用运动感知算法精准区分“坐定”与“起身”， 并以纯 ASCII 界面直观反馈久坐时长与健康状态。代码结构清晰， 参数可调，便于二次开发与个性化适配。 该设备可广泛应用于长期伏案工作者、 学生等需要定时起身活动的场景，有效降低久坐带来的健康风险。
