# code2things-zhou-tao

## Overview

This is a static website project (examples/exercises/portfolio) used to keep a course learning log and front-end practice. The repository contains example pages, styles, images, and video resources for learning and quick demos.

## Key Folders

- `html/`: site pages and examples (includes `index.html`).
- `css/`: stylesheets (contains Bootstrap files).
- `js/`: JavaScript files (contains Bootstrap scripts).
- `images/`: image assets.
- `project_html/`: example project pages (e.g. `Project1.html`, `Project2.html`).
- `weekly_html/`: weekly exercises or class notes.
- `videos/`: recorded videos.

## Running Locally

The project is purely static. Open `html/index.html` in your browser to view it. To serve via a simple local HTTP server (recommended for testing):

```
python -m http.server 8000
# then open http://localhost:8000/html/index.html in your browser
```
